# Money Lending - Mobile App

A mobile application for managing money lending and borrowing records. Built with React Native, Expo, and TypeScript.

![React Native](https://img.shields.io/badge/React_Native-Expo-61DAFB?logo=react&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6?logo=typescript&logoColor=white)
![Expo](https://img.shields.io/badge/Expo-SDK-000020?logo=expo&logoColor=white)

## Features

- Track money lent and borrowed
- Mobile-first design with NativeWind (Tailwind CSS for React Native)
- Cross-platform (iOS & Android)
- Built with Expo for easy development and deployment

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React Native + Expo |
| Language | TypeScript |
| Styling | NativeWind (Tailwind CSS) |
| Build | EAS Build |
| Bundler | Metro |

## Getting Started

### Prerequisites

- Node.js >= 18
- Expo CLI (`npm install -g expo-cli`)
- Expo Go app on your phone (for testing)

### Installation

```bash
# Clone the repository
git clone https://github.com/akashffjdnh/Money_lending.git
cd Money_lending

# Install dependencies
npm install

# Start Expo development server
npx expo start
```

Scan the QR code with Expo Go (Android) or Camera app (iOS) to run on your device.
